Formulas

Reactance for the capacitor at frequency. X c = 1 2 ∗ π ∗ f ∗ c

Impedance for the circuit. Z = X c 2 + R 2

Output frequency can be found with this formula. V o u t = V i n ∗ X c Z

Gain is calculated with this formula. G db = 20 log ⁡ V out V in

Phase shift formula. ϕ = − a r c t a n ( 2 π f r c )
