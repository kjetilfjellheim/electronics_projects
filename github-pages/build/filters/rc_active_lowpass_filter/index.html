Formulas

Calculates minimum 60db attenuation at frequency Fs. The attenuation result is the total order number of the circuit. A s = f s f 3 d b

To calculate the resistor values in the circuit. Rxn is the normalized normalized value = 1. R x = Z × R x n

To calculate the capacitor values use the value for the capacitor and calculate the capacitance the capacitors is to use in the circuit. C x = C t a b l e Z × 2 × π × f 3 d b
