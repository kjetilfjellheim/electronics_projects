555 metal detector

Description

The 555 will oscillate based on the LCR circuit. When metal comes in contact with the inductor the oscillation will change due to it affecting the magnetic field around the inductor. The output is the sent to the speaker which will change tone when metallic (Magnetic) objects comes near.

This is not a particularly powerful metal detector, for that a greater voltage/current is requred for the inductor so that it generates a larger magnetic field. Using a coil rather than a simple inductor would also make the metal detector better as it would cover a larger area.
